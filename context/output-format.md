# Discord 출력 포맷 가이드 (공통)

모든 크론 태스크의 Discord 출력은 이 파일의 규칙을 따른다.

## 기본 카드 구조

```
━━━━━━━━━━━━━━━━━━━━
[이모지] MM-DD (요일) [태스크명]
━━━━━━━━━━━━━━━━━━━━
한 줄: [상황 요약 — 주니어 팀원 톤, 구어체]

[🔴 긴급 항목들 — 지금 봐야 할 것. 없으면 이 섹션 통째로 생략]
[🟡 참고 항목들 — 나중에 봐도 됨. 없으면 통째로 생략]

[이모지] [카테고리]  [값]
[이모지] [카테고리]  [값]
...
━━━━━━━━━━━━━━━━━━━━
```

## 작성 규칙

### 한 줄 요약
- **주니어 팀원 톤**: "오늘은 좀 조용했어요" / "TQQQ가 좀 흔들리고 있어요, 확인해보세요"
- 구어체 + 살짝 친근함. 딱딱한 보고체 금지.
- 가장 중요한 1가지를 한 문장에 담는다.

### 중요도 분류
- **🔴 긴급**: CRITICAL 장애, 손절선 근접, P0 이슈 → 없으면 섹션 생략
- **🟡 참고**: YELLOW 경고, 오늘 중 처리 필요한 것 → 없으면 섹션 생략
- 🟢 정상 항목은 데이터 행에서 "정상" 한 단어로 끝냄. 별도 섹션 없음.

### 데이터 행
- 이모지 + 카테고리(고정폭 정렬) + 값
- 정상이면: `💻 시스템   정상`
- 이슈면:   `💻 시스템   디스크 87% ⚠️`
- 없는 데이터는 행 자체 생략

### 금지사항
- 마크다운 섹션 헤더(###) 사용 금지
- "정상입니다 ✅", "이상 없습니다" 같은 반복 문구 금지
- 빈 섹션 출력 금지 (데이터 없으면 행 생략)
- "저장했습니다", "Sources:" 등 메타 텍스트 금지
